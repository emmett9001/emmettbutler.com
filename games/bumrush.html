Bum Rush

Downloads: Native Mac build, AIR Package (Mac/Windows, requires Adobe AIR)

Bum Rush is an eight player local multiplayer car combat dating sim racing game about 8 college roommates who all have sexy dates on the same night. Players race, bump, and nab dates while trying to get home first to claim the dorm for some private time.

This game requires one USB controller per player. It supports PS3, PS4, and XBox 360 USB controllers. This is the controller model used during No Quarter.

Originally made for the No Quarter exhibition of games in NYC - October 9, 2015 - Emmett Butler, Nina Freeman, audio by Maxo, art by Diego Garcia.

This game was made using Flixel. You can view the Github repository.
