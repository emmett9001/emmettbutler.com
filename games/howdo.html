how do you Do It

Click in the above box to play

Originally made for Global Game Jam, Jan 24-26 2014 - Emmett Butler, Nina Freeman, audio by Deckman Coss, art by Jonathan Kittaka.

how do you Do It was nominated for Best Audio and Best Overall at the NYC 2014 Global Game Jam site.

This game was made using Flixel, box2d, Photoshop, the GNU Image Manipulation Program, Reaper, and Renoise. You can view the Github repository.
