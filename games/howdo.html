how do you Do It?

Click in the above box to play

Originally made for Global Game Jam, Jan 24-26 2014 - Emmett Butler, Nina Freeman, audio by Decky Coss, art by Joni Kittaka.

how do you Do It was nominated for Best Audio and Best Overall at the NYC 2014 Global Game Jam site. It was an Indiecade 2014 Finalist and an IGF 2015 Nuovo Finalist, and is available on Steam.

This game was made using Flixel, box2d, Photoshop, the GNU Image Manipulation Program, Reaper, and Renoise. You can view the source code in the Github repository. You can also listen to an extended version of the background music by Decky Coss here.
